Ringkasan Alur Pengguna

Sembilan layar inti dari perjalanan pengguna Magna Via — dari pembukaan narasi hingga konsultasi akhir bersama Cenayang. Setiap layar mencerminkan estetika tarot-RPG berlatar malam berbintang Arcadia.

  1. Splash — gerbang Arcadia
  2. Opening Narrative — Oracle memperkenalkan dunia
  3. Biodata Form — pendaftaran ke Dewan Kerajaan
  4. Hobby Card Selection — pilih tiga jalur
  5. Birth Star Selection — pilih elemenmu
  6. Quiz Scenario — skenario cerita pilihan
  7. Result Reveal — transisi seremonial
  8. Result — pengungkapan kelas karakter
  9. Cenayang Chat — konsultasi bersama Oracle
